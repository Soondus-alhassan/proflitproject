PROFILit is an 'in-house' data profiling tool developed by the team shown above. Unlike it's rivals, PROFILit can profile data in minutes with descriptive reports allowing you to visualize the quality of your data.

The tool began as an application which was later converted to this web application for a number of reasons:

  • Ease of access
  • Improved aesthetics
  • Additional functionality such as the ability to add and delete files within the app.
